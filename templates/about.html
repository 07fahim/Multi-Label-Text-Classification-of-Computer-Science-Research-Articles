About the Project

A comprehensive multi-label classification system for Computer Science research articles

Project Overview

This project classifies arXiv papers into multiple Computer Science subjects using their abstracts. Built with state-of-the-art NLP techniques, it leverages SciBERT's domain-specific knowledge to achieve high accuracy in predicting research paper categories.

Key Achievements

  • üìä Scraped and processed ~30,000 papers from arXiv (2023-2025)
  • üß† Trained and evaluated multiple transformer models
  • üéØ Achieved 97.43% accuracy with SciBERT
  • ‚ö° Optimized model size by 74% using ONNX quantization
  • üöÄ Deployed on multiple platforms for accessibility
